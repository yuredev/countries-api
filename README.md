## Creating countries:


```
https://tads-countries-api.herokuapp.com/countries/
```


```json
{
  "name": "Brasil",
  "capital": "Brasilia",
  "currency": "Real",
  "language": "Portugues",
  "population": 210000000,
  "area": 8000000
}
```

## Updating countries:

```
https://tads-countries-api.herokuapp.com/countries/
```

{
  "name": "Republica Federativa do Brasil",
